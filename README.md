## ℹ️앱 설명

소리를 녹음 및 재생할 수 있는 녹음기 앱

<br>

## ✅구현 기능

* CustomView를 활용하여 소리 시각화
* MediaRecorder와 MediaPlayer를 활용하여 녹음 및 재생 기능 구현

<br>

## ✅사용되는 기능

* Android
    * MediaPlayer
    * MediaRecorder
    * Permission Request
    * Canvas
    * Handler
    * CustomView
